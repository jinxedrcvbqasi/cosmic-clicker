# cosmic-clicker v2 22.03.26

🆕 What’s Added in v2
🎮 Gameplay
🏅 16 Achievements — from “First Contact” to “Cosmic Tycoon”, pop-up notifications unlock in real-time
♻️ Prestige System — unlocks at 1M coins, resets progress, grants permanent multipliers ×1.5 / ×2.0 / ×2.5…
📋 Daily Quests — 3 tasks per day (clicks / coins / upgrades), reset at UTC midnight, scale with prestige
☄️ Random Events — occur every 6–14 minutes: Meteor Shower, Click Frenzy ×3, Auto Surge ×5, Lucky Strike, with progress bar in the header
👥 Social Features
💬 Real-Time Chat — global chat, last 35 messages, rate limit 2.5s
⚔️ Clans — create for 500 coins, join for free, clan leaderboard by total coins
🥊 Duels — challenge online players, 30-second click battles, winner earns coins
🎁 Gifts — send coins to any online pilot
✨ Visuals & UX
💥 Particle Effects — canvas particles on every click, bigger & brighter for crits
🔊 Sounds — Web Audio API: click, crit, purchase, achievement, prestige, event, gift (toggle 🔊/🔇 in header)
🎨 8 Coin Skins — 🪙💎⭐🌕🔮💰☀️🌀, some unlockable with coins
📈 Coin Graph — canvas bar chart, updates every 30 seconds
