# cosmic-clicker v4 23.03.26 https://jinxedrcvbqasi.github.io/cosmic-clicker/

🚀 Project Update Changelog

Here is a clear breakdown of what was added in each major version.

---

🌌 Version 3 — Gameplay Expansion

This update focused on making the game feel more alive and rewarding over time.

🌍 Global Raids

- A daily boss planet appears in the galaxy.
- All players can attack it together.
- When the boss is defeated, everyone receives rewards based on contribution.

💤 Offline Income

- The game now generates coins while you are offline.
- Works for up to 8 hours.
- When you return, a popup shows how many coins were earned.

⚗️ Crafting System

- Players can collect resources from events.
- Resources can be used to craft boosts and cosmetic skins.

📲 PWA Support

- The game can now be installed on a phone like an app.
- Works directly from the home screen.

🛡 Anti-Cheat Improvements

- Rate limiting for actions.
- Server validation of player scores using Firebase.

🔔 Notifications

- Notifications for important events:
  - raids starting
  - duel invites
  - special events

🗺 Galaxy Map

- Added a visual galaxy map.
- Contains multiple planets with different mechanics.

🎰 Mini-Games

- New mini-games to earn coins:
  - Space Slots
  - small arcade-style mechanics

---

🌌 Version 4 — Progression & World Systems

This update added deeper progression systems and expanded the game universe.

🔐 Account System

- Added Google Sign-In.
- Anonymous login also available.
- Player progress is now tied to Firebase Auth UID instead of a nickname.

🌳 Technology Tree

A full research system with 4 branches:

- Click Power
- Automation
- Economy
- Prestige

Includes 16 technologies that provide permanent bonuses like:

- stronger clicks
- faster auto income
- critical click chance
- better offline rewards
- trading discounts

🏰 Clan Base System

Clans can now build and upgrade a shared base.

Available buildings:

- Power Core
- Research Lab
- Shield Generator
- Trading Post
- Nebula Tap

Clan members contribute coins to a shared bank, and upgrades provide bonuses to all clan members.

⚗️ Alchemy & Artifacts

Added a crafting system for rare artifacts.

Examples:

- Phoenix Core
- Void Heart
- Star Crown

Players can equip up to 2 artifacts at once for powerful permanent bonuses.

📖 Story / Lore

The galaxy now has a storyline.

- 8 unlockable chapters
- chapters unlock as the player progresses
- reveals the history of the universe

🎭 Seasonal System

The game now has weekly seasons.

Each season:

- introduces a new galaxy theme
- changes UI colors
- gives special bonuses
- includes exclusive seasonal rewards

Example themes:

- Solar Flare
- Ice Nebula
- Volcanic Core

Players can earn rewards by reaching 500 seasonal points.

🤖 NPC Traders

Random NPC traders can appear in the game.

They offer special deals such as:

- resource exchanges
- temporary boosts
- rare items

They appear randomly every 15–25 minutes.

🌍 Multi-Language Support

The game now supports multiple languages.

Available languages:

- Russian
- English

Players can switch languages directly from the interface.

---

✨ More updates are planned in future versions.
